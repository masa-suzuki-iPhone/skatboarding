# :iphone:About App
<p> 私は2021年の3月に「SkateSpot」というアプリをAppStoreにリリースしました。<br>「SkateSpot」は、全国の練習場所に困っているスケーターのためのアプリです。全国にいるスケーター同士でスケボーのできるスポットをシェアする事ができます。さらに、投稿されたスポットにコメントやいいねもする事ができ、ユーザーの感想を把握する事もできます。</p>
